# Faculty Workflow Example

## Scenario
A faculty member wants to use AI to create a low-stakes quiz for Week 1 of an XR and Social Engagement course.

## Bad approach: zero-shot
Prompt:
> Write a quiz on XR.

## Better approach: grounded workflow
Use:
- syllabus
- Week 1 module materials
- course notes on the value and purpose of the course
- documents about murals in New Brunswick
- notes about phone-based or hybrid immersive design

Then prompt with grounded context and review the result.
