# Start Here

This cookbook is not meant to be read front to back like a book.

Use it when you have a concrete teaching task and want AI help that stays grounded in your course.

## How to use this cookbook
1. Start with your task
2. Gather your real materials
3. Pick one recipe
4. Use the recipe prompt
5. Review the result

## Most important rule
Do not rely on zero-shot behavior for important teaching tasks.
