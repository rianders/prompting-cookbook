# Local AI Option

## What it is

Local AI tools run language models entirely on your own machine — no internet connection, no cloud transmission, no third-party data handling.

**LM Studio** is the recommended starting point for faculty who want to experiment locally.
→ [lmstudio.ai](https://lmstudio.ai/)

## When to use it

- Developing and refining prompts before using them in an approved institutional tool
- Experimenting with models or settings that institutional tools do not expose
- Working with materials you prefer not to send through any cloud service
- Testing prompt recipes from this cookbook in a fully contained environment

## Benefits

- **No cloud transmission** — your materials stay on your machine
- **No institutional restrictions** — try any available model freely
- **Safe prompt testing** — iterate on prompts without data-handling concerns

## Limitations

- Requires a capable local machine (RAM and GPU matter for larger models)
- Models available locally may differ in quality from cloud-hosted models
- Not a substitute for approved tools when sharing outputs or working within official workflows
