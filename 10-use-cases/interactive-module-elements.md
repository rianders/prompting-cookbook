# Interactive Module Elements

## Use case
Use AI to generate small interactive practice elements inside or alongside a Canvas module.

## Good fits
- vocabulary practice
- low-stakes self-checks
- short branching or decision activities
- quick review games or matching exercises

## What to gather first
- module objective
- terms, concepts, or skills students should practice
- student level
- accessibility requirements
- hosting or Canvas constraints

## Design principle
The interaction should make the learning task clearer or more engaging. Do not add motion or interactivity that does not improve the instructional sequence.

## Watchouts
- if the activity needs hosting, plan that constraint before prompting
- keep first versions small and low-stakes
- require a completion or summary artifact if students may need to submit proof of completion
