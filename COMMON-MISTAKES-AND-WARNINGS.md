# Common Mistakes and Warnings

## 1. Using zero-shot grading
Provide rubric, learning objectives, assignment description, and module materials.

## 2. Asking for a result without supplying the source materials
Attach syllabus, readings, assignments, examples, and notes.

## 3. Forgetting time context
Supply current semester materials or specify the historical frame you are teaching.

## Core warning
Fluent output is not the same as course-aligned output.
